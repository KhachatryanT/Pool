Этот проект содержит в себе клиентский код и ссылку на Pool.Api, откуда подтягиваются все необходимые ресурсы, включая appsettings.json, nlog.config, etc.
Это необходимо для минимилизации связанности между клиентским слоем и веб-апи, но, едином хостинге на dotnet платформе для экономии ресурсов.
В дальнейшем, клиентский слой может быть переведен на другой хостинг.
